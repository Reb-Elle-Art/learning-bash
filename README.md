# learning-bash

A personal bash reference built during learning sessions on Ubuntu 24.04 LTS (Enid) accessed via Tailscale SSH from macOS Tahoe.

## Contents

- **index.html** — Interactive bash skills reference with four tabs:
  - Commands (filterable by category, expandable with examples and flags)
  - Building blocks (path symbols, wildcards, operators, scripting)
  - Flags (common flags and which commands use them)
  - Concepts (the bigger ideas behind the commands)

- **bash_concepts_tracker.html** — Earlier version of the concepts tracker

## Viewing

Hosted via GitHub Pages at:
https://reb-elle-art.github.io/learning-bash/

Or clone and open locally:
```bash
git clone https://github.com/Reb-Elle-Art/learning-bash.git
cd learning-bash
open index.html
```

## Setup context

- Local machine: macOS Tahoe 26.3.1, Apple Silicon, zsh
- Remote VM: Ubuntu 24.04.4 LTS (Noble Numbat), bash
- Networking: Tailscale
- Web server: nginx
- Editor: Helix
