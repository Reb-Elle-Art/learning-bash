Concepts
Navigation
pwd
Print working directory -- where am I?
ls
List contents of current directory
ls -a
List including hidden files (dotfiles)
ls -lh
List with sizes in human readable format
ls -lah
Full detail view including hidden files
cd <dir>
Change directory -- move into a folder
cd ..
Go up one level to parent directory
Files & directories
cat <file>
Print file contents -- good for short files
less <file>
Scrollable viewer. q to quit
head <file>
Show first 10 lines
tail <file>
Show last 10 lines
wc -lw <file>
Count lines and words in a file
file <file>
Show what type of file something is
find ~ -name
Search for a file by name
find . -exec
Find files and run a command on each
grep
Search for text inside files
mv <src> <dst>
Move or rename a file or directory
cp <src> <dst>
Copy a file. Use -r for directories
rm -r <dir>
Delete a directory and its contents
mkdir <name>
Create a new directory
du -sh <dir>
Show size of a directory in MB/GB
pandoc
Convert between file formats (e.g. rtf to md)
Remote & SSH
ssh user@host
Connect to a remote machine
exit
Disconnect from SSH session
tailscale status
List devices in your Tailscale network
scp <src> <dst>
Securely copy files between machines
scp -r
Securely copy entire directories
scp host:path/*
Copy contents of a remote directory
System & admin
hostname
Show which machine you are on
echo $SHELL
Show which shell you are using
which <program>
Show where a program is installed
sudo <cmd>
Run a command as administrator
sudo passwd root
Change the root password
lsb_release -a
Show Ubuntu version info
brew install
Install software on Mac via Homebrew
apt install
Install software on Ubuntu
apt update && upgrade
Update and upgrade packages on Ubuntu
systemctl status
Check the status of a background service
sudo reboot
Restart the machine
Key concepts
CLI
Command Line Interface -- text-based way to interact with a computer
Shell
The program that reads and runs your commands (bash, zsh)
Flag / option
Modifies a command's behavior. Starts with - (e.g. ls -a)
Wildcard *
Means "anything" -- e.g. *.md matches all markdown files
Pipe |
Sends output of one command as input to another
; separator
Run this command then the next, no matter what
&& separator
Run the next command only if the first succeeded
$() substitution
Use the output of a command as a value, e.g. $(find . -name "*.txt")
sudo
Superuser do -- temporarily run a command as admin
. and ..
. = current directory, .. = parent directory
~ (tilde)
Shortcut for your home directory
Absolute vs relative path
Absolute starts with / (from root). Relative starts from where you are
Hidden files
Files starting with . are hidden by default (e.g. .openclaw)
Git / .git
Version control system. .git folder stores change history
Bash scripting
Chaining commands with logic like loops (for f in *.rtf...)
LTS
Long Term Support -- stable, long-supported software version
Markdown
Simple text formatting language, files end in .md
Package manager
Tool for installing software. brew on Mac, apt on Ubuntu
nginx
A web server -- serves files to a browser over a network
systemctl
Manages background services on Ubuntu (start, stop, status)
Prompt symbols
$ = regular user (bash), % = regular user (zsh), # = root/admin
.bak convention
Renaming a file with .bak marks it as a backup -- still there but ignored
